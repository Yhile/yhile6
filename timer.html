How to use the timer

  1. Choose Countdown for a timer that counts down to zero, or Stopwatch to count up from zero.
  2. For countdown: enter hours, minutes, and seconds — or click one of the quick-set chips (1 min, 5 min, 25 min, etc.).
  3. Click Start. The timer runs even if you switch tabs. Click Pause to pause and Resume to continue.
  4. When the countdown reaches zero, a beep alarm plays automatically. Click Reset to start again.
  5. For the stopwatch, click Lap while running to record lap times with individual split times.

Frequently asked questions

Does the timer play a sound when it finishes?
Yes. When the countdown reaches zero, the timer plays a three-beep alarm using the Web Audio API. No internet connection, plugin, or sound file download is required — it's generated in your browser.
Does the timer keep running if I switch tabs?
Yes. The countdown uses the system clock rather than just counting intervals, so it stays accurate even when the tab is in the background.
Can I record lap times with the stopwatch?
Yes. Click the Lap button while the stopwatch is running. Each lap entry shows both the total elapsed time and the individual split time for that lap.
What is a Pomodoro timer?
The Pomodoro Technique uses 25-minute work sessions followed by 5-minute breaks. Click the 25 min chip to instantly set a Pomodoro work session, and 5 min for a short break.
Is this timer free?
Yes — 100% free, no signup, no account, no ads. yhile.com is free forever.
